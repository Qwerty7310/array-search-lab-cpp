# array-search-lab-cpp

Небольшой учебный проект на C++ для сравнения алгоритмов поиска в массиве.

Что здесь есть:
- линейный поиск (`Better_Linear_Search`)
- линейный поиск с барьером (`Sentinel_Linear_Search`)
- поиск в упорядоченном массиве (`Ordered_Array_Search`)
- бинарный поиск (`Binary_Search`)

Программа запускает эти алгоритмы на массивах разного размера и выводит количество сравнений для поиска элемента в начале, середине и конце массива.

## Сборка и запуск

```bash
cmake -S . -B build
cmake --build build
./build/array-search-lab-cpp
```

Проект сделан как лабораторная работа: цель не «идеальная библиотека поиска», а наглядное сравнение подходов.
